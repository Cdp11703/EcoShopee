# EcoShopee
A compilation of various UI prototypes in a Figma Workspace for an economically aware shopping app. Users on E-commerce platforms need to be made more aware and knowledgeable about how to responsibly handle products with potentially hazardous materials that they purchase as it would result in less health hazards, damage to the environment, and also a better experience with the products.

The final prototype that we have presented in 2024 is the canva link.
